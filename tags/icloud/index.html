How to access iCloud Drive from terminal

Access iCloud Drive from terminal Having an iCloud account allows keeping files in sync, despite which machine one is working on. As it is very straightforward to manage folders and files via Finder, it is rather cumbersome to access these items via terminal. If you are a developer it is highly likely you are using terminal often, to develop, create config files, write notes and more. It is surely useful to keep all this files in the cloud.
